# Тестовое задание 1:
Реализация наиболее точного таймера

## Языки:

- HTML
- CSS
- JavaScript

## Сборка проекта

Чтобы собрать и запустить проект на [localhost:3000](http://localhost:3000) выполните следующие команды в своей локальной директории:
- `$ git clone `
- `$ npm install`
- `$ npm run build`
- `$ npm run dev`
